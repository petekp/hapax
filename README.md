# Hapax

A curated visual collection of rare words—each styled with a unique font and color palette.

## Development

```bash
pnpm install
pnpm dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Production build |
| `pnpm seed-gallery` | Populate gallery with styled words |

## Stack

Next.js 16 · React 19 · Tailwind CSS 4 · Framer Motion
