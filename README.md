# Hapax

A gallery of rare words, each with its own font and color.

## Development

```bash
pnpm install
pnpm dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Production build |
| `pnpm seed-gallery` | Populate gallery with styled words |

## Stack

Next.js 16 · React 19 · Tailwind CSS 4 · Motion
